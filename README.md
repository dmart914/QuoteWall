# QuoteWall
A web homepage consisting of an image, a font and a quote.

## Architecture
* Front-end
  * AJAX calls
    * Quote
    * Image location
    * Font
  * Link to login
* Authentication
  * Add quote
  * Upload image
  * Upload font file
* REST API
  * GET/POST Image Upload
  * GET/POST Font file Upload
  * GET/POST JSON Quote text
